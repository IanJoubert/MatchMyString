MatchMyString ReadMe

MatchMyString is a UWP app used to find similar results in a 
file based in a provided search string.

MatchMyString can be run on any Microsoft device running the 
Windows 10 Fall Creators Update (Build 16299) or later which 
has developer mode activated.
see: https://docs.microsoft.com/en-us/windows/uwp/get-started/enable-your-device-for-development


The app has been prepackaged so all you need to do is 
install the app by running the 

Mms.Presentation.App_1.0.0.0_x86_x64_arm.appxbundle 

packaged bundle which can be found at: 

MatchMyString\Mms.Presentation.App\AppPackages\Mms.Presentation.App_1.0.0.0_Test

You may also need to install the 
Mms.Presentation.App_1.0.0.0_x86_x64_arm.cer certificate 
under the Trusted Root Certification Authorities store. 
see https://docs.microsoft.com/en-us/windows-hardware/drivers/install/installing-test-certificates 


Alternatively, the app can be run using Visual Studio 17 by opening the solution file and starting the app.
